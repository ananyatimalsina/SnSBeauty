# SnSBeauty
Full Stack appointment management Web-App for <a href="https://www.instagram.com/snseyebrow/">S&S Eyebrow<a/>, an eyebrow threading business located in Palmdale, California.

## Stack
Frontend: React<br></br>
Backend: Django<br></br>
Phone Number Verification: <a href="https://www.twilio.com/docs/usage/api">Twilio<a/>
